Typography

Headings should in Boogaloo font 400 weight, no additional styling. H2 should be main color. paragraphs are in roboto font 400 weight. For desktops heading size should be 3rem for h1, 2.5rem for h2, 2rem for h3. Sizes should scale for smaller screen resolutions. Sizes should scale for smaller screen resolutions. Paragraph text should be in desktop, use Ultra-bold or light for distiguishing.
